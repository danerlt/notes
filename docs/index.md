## 2. 为什么要用mkdocs

mkdocs是一个用于编写文档的工具，它的优点有：

1. 有很多优秀的主题可供选择
2. 自定义配置简单
3. 可以编书写边预览，预览速度比vuepress快多了
4. 方便的部署到github pages
5. 使用 Python 编写, 可以方便的扩展
6. 支持多种格式的问题，例如md，ipynb, py等(ipynb和py格式需要安装插件)
