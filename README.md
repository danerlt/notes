## 2. 为什么要用mkdocs
mkdocs是一个用于编写文档的工具，它的优点有：
1. 生成的网页简洁，不会有太多的干扰
2. 生成的网页可以在手机上阅读
3. 生成的网页可以在离线状态下阅读
4. 生成的网页可以方便的导出为pdf
5. 使用 Python 编写, 可以方便的扩展
